In this assigment, I implemented cube maps.
I created an environment map using a single tiff image.
I simulated a reflective teapot using by passing the reflected rays into the cube map.

description of GUI:
- DBG: starts up the scene

Moving the camera using keys on the keyboard:
- w: revolve up
- s: revolve down
- a: revolve to the left
- d: revolve to the right
- left arrow key: roll counter-clockwise
- right arrow key: roll clockwise
